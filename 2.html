Firefox

刷新 Firefox —— 重置附加组件和设置

（重定向自重置 Firefox —— 轻松解决大多数问题）

此功能仅桌面和笔记本电脑上的 Firefox 中可用。

此功能在最新版 Firefox 中已得到改进。更新 Firefox 以使用改进后的版本。

如果您在使用 Firefox 时遇到问题，刷新 Firefox 可以帮您解决问题。刷新功能可将 Firefox 还原到出厂的默认状态以解决问题，并且保留书签、密码、当前打开的标签页等必要信息。
注意： 扩展 和扩展数据会丢失。

刷新 Firefox

  1. 如果你正用 Firefox 阅读本页面，请直接单击按钮 刷新 Firefox。
    如果是其他移动浏览器，该操作无效。
    • 你还可以在 about:support 故障排除信息 页面顶部找到刷新 Firefox 的按钮。
  2. 接下来，在弹出的确认窗口中点击 刷新 Firefox。
  3. Firefox 将自动关闭以完成刷新。刷新结束后，一个窗口会列出您的重要信息。点击完成，Firefox 将打开。


注意：如果您无法正常启动 Firefox，在 Firefox 安全模式 的窗口中也有一个刷新 Firefox 按钮。如果你没能找到 Firefox 的刷新按钮，你还可以通过新建一个配置文件来手动刷新。请参看 如何新建配置文件。

刷新功能如何工作？

你的所有 Firefox 设置和个人信息都存放在配置文件夹中。刷新功能会创建一个新的用户配置文件夹，并保存您的重要数据。

存储在 Firefox 配置文件夹的附加组件，比如扩展和主题，这些将被移除。存储在其他位置的附加组件，比如插件，不会被移除，但是所有的设置（比如你禁用的插件）会被重置。

Firefox 将保留以下项目：

  • 书签
  • 浏览历史
  • 密码
  • 打开的窗口、标签页组标签页
  • Cookie
  • 自动填表信息
  • 用户字典

以下项目或设置将被删除：

  • 扩展和主题、网站权限、修改过的首选项、添加的搜索引擎、下载历史、DOM 存储、安全证书和设备设置、下载动作、插件设置、工具栏定制、用户样式和 社交服务 将被移除。
注意：以前的 Firefox 配置文件将放在桌面上一个叫做“旧的 Firefox 数据”的文件夹里。如果重置刷新没能修复问题，您仍可将一些未被保留的信息复制到新的配置文件夹中。详情请参见在配置文件夹之间复制文件。因为该文件夹包含了您的敏感信息，所以当您不再需要这个文件夹时应删除它。
